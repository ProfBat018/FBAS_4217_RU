git init - инициализирует пустой локалльный репозиторий 
git remote - набор команд, которые нужны для работы с макросом наших ссылок.
origin - назвние переменной, которая хранит ссылку на наш репозиторий.
git checkout -b  - создает ветки 
git branch - проказывает список всех ваших локальных веток. 
git add . - добавляет все файлы файлы в очередь.
git commit -m "ваше сообщение" -  подтверждает все изменения, которые были сделаны после git add .
git push origin Название ветки - добавляет изменения на удаленный репозиторий 
git rm .\ConsoleApp1\ -r - Удаляет файл с ветки
