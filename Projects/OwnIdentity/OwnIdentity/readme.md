# Про пароли 

### Для начала нам надо знать что шифронавание и хеширование.

Шифрование - это процесс преобразования информации в нечитаемый вид, который называется шифротекст. Шифрование используется для защиты информации от несанкционированного доступа, а также для обеспечения безопасности передачи данных по открытым каналам связи.

Хеширование - это процесс преобразования информации в хеш-код, который называется хеш-сумма. Хеширование используется для проверки целостности данных, а также для обеспечения безопасности передачи данных по открытым каналам связи.
