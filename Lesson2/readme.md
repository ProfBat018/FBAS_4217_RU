## Есть два типа тегов
* Одинарные теги - здесь link и meta - одинарные теги
* Двойные теги - здесь html двойной тег

```
<html>
<link rel="stylesheet" href="style.css">
<meta charset="utf-8">
</html>
```
.

## Есть два типа элементов
* Блочные элементы
* Инлайновые элементы(строчные)
```
<div>
    <p>Это блочный элемент</p>
    <span>Это строчный элемент</span>
</div>
```

## Tags
* h1-h6. Это заголовки. h1 - самый главный, h6 - самый маленький.
* p - параграф
* a - ссылка
* img - картинка
* ul - неупорядоченный список
* ol - упорядоченный список
* li - элемент списка
* div - блок
* span - строчный элемент. Используется для выделения части текста
* i - курсив или иконка 
* table - таблица
* tr - строка таблицы
* td - ячейка таблицы
* th - ячейка таблицы с заголовком

## Что же такое cdn и зачем он нужен ?

**CDN** - это сеть серверов, которые расположены по всему миру. Они хранят копии контента, 
который может быть доставлен посетителю сайта.
![Шо такое CDN ?](https://blog.xeonbd.com/assets/uploads/2017/07/how-content-delivery-network-works.png)

**defer** - атрибут, который говорит браузеру, что он должен продолжать обрабатывать страницу и загружать скрипт в фоновом режиме, а затем запустить этот скрипт, когда он загрузится.

## Ключевые слова для задания **размера**
* px - пиксели
* em - относительные единицы измерения. Относительно размера шрифта родительского элемента
* rem - относительные единицы измерения. Относительно размера шрифта корневого элемента
* % - проценты
* vw - viewport width. 1vw = 1% ширины вьюпорта
* vh - viewport height. 1vh = 1% высоты вьюпорта
* vmin - viewport min. 1vmin = 1% меньшей стороны вьюпорта. ***viewport - это область, в которой виден сайт***
* vmax - viewport max. 1vmax = 1% большей стороны вьюпорта.
* ch - ширина символа "0" в шрифте
* ex - высота символа "x" в шрифте
* cm - сантиметры
* mm - миллиметры
* in - дюймы
